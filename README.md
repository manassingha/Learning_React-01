# React Hero Section Website

A modern React + Vite frontend project featuring a responsive hero section with navigation bar, branding icons, and clean UI design.

## 🚀 Features

- Responsive Navbar
- Hero Section Layout
- Brand Icons Section
- React Component Structure
- Fast Development using Vite
- Clean CSS Styling
- Reusable Components

## 🛠️ Tech Stack

- React JS
- Vite
- JavaScript
- CSS3
- ESLint
